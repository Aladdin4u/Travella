## Installation

1. Clone repo

### Backend installation

1. cd api
1. Set up environment variables
   ```env
   cp .env.example .env
   ```
1. Enter your data into .env
1. run `yarn install`
1. run `yarn dev`
1. navigate to https://localhost:8000

### fontend installation

1. cd client
1. Set up environment variables
   ```
   cp .env.example .env
   ```
1. Enter your data into .env
1. run `yarn install`
1. run `yarn dev`
1. navigate to https://localhost:3000

### Admin Panel installation

1. cd admin 
1. Set up environment variables
   ```
   cp .env.example .env
   ```
1. Enter your data into .env
1. run `yarn install`
